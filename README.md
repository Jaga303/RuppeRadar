# Rupee Radar 💰📊

Rupee Radar is an Android-based personal finance management application that helps users track expenses, manage budgets, analyze spending patterns, and gain insights into their financial habits. The app provides an intuitive interface for recording transactions and monitoring financial activities efficiently.

## Features

* 📌 Track income and expense transactions
* 📊 Visualize spending with analytics and reports
* 💰 Set and manage monthly budgets
* 🔔 Expense notifications and reminders
* 🗂️ Categorize transactions for better organization
* 📈 View financial summaries and insights
* 🏠 User-friendly dashboard for quick access

## Tech Stack

* Java
* Android Studio
* Room Database
* SQLite
* Material Design Components
* RecyclerView
* Fragments & Navigation

## Installation

1. Clone the repository:

```bash
git clone https://github.com/your-username/RupeeRadar.git
```

2. Open the project in Android Studio.
3. Sync Gradle dependencies.
4. Build and run the application on an emulator or Android device.
